1. Описание проекта:

Этот проект программа на языке Java для работы с продуктами из файла products.txt. В ней есть два класса: Main и Products. 
Класс Main содержит метод main, который является точкой входа в программу и отвечает за чтение данных о продуктах из файла, 
создание объектов класса Products и вывод информации о продуктах в виде таблицы в консоль. Класс Products представляет отдельный 
продукт и содержит поля и методы для работы с ними. У каждого продукта есть идентификатор, название, цена, бренд и количество. 
В классе Products определены методы для клонирования и сравнения объектов типа Products, а также для чтения данных из файла 
и вывода информации о продуктах в различных форматах.


2. Для успешного запуска проекта, необходимо иметь:

1. JDK (Java Development Kit) установленный на компьютере.
2. Файл products.txt с данными о продуктах, расположенный по пути "D:/Soft/IntelliJ IDEA Community Edition 2022.3.3/projects/lab0/src/products.txt".
3. IntelliJ IDEA
4. Нужно запустить класс Main в IntelliJ IDEA


3. Описание запуска приложения:

1. Открыть проект в IDE (IntelliJ IDEA) и убедиться, что работают стандартные библиотеки Java.

2. Запустить проект. Для этого нужно открыть файл Main.java и нажать на кнопку "Run" в IDE 

3. Убедится, что нужные файлы находятся в нужном месте. 
В нашем случае, файл products.txt должен находиться в пути D:/Soft/IntelliJ IDEA Community Edition 2022.3.3/projects/lab0/src/.
